STAR TREK COMPUTER — SOUND GUIDE
=================================

Replace the .wav files in each folder with Star Trek sounds.
Free sources: freesound.org, trekcore.com/audio

GREETING SOUNDS (startup): 
- TNG computer startup chime
- "LCARS system online" type sounds
- Suggestion search: "star trek computer startup wav"

ACK SOUNDS (I heard you):
- Short beep/chirp (like pressing a LCARS panel)
- Suggestion search: "star trek lcars beep wav"

THINKING SOUNDS (processing loop):
- Soft processing hum or data transfer sound
- Suggestion search: "star trek computer processing wav"  

ERROR SOUNDS:
- Red alert style short beep
- Suggestion search: "star trek alert beep wav"

FORMAT: All files must be .wav format, mono or stereo, any sample rate.
